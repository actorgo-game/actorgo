# gin组件
- 基于gin实现http server

## Install

### Prerequisites
- GO >= 1.17

### Using go get
```
go get github.com/actorgo-game/components/gin@latest
```


## Quick Start
```
import cgin "github.com/actorgo-game/components/gin"
```
